# Overly Dramatic Command-Line Calculator
## Introduction
The Overly Dramatic Command-Line Calculator is a Python application designed to perform basic arithmetic operations (addition, subtraction, multiplication, and division) on two numbers. The calculator expresses reluctance and dissatisfaction with performing math operations, using dramatic language to convey its 'distaste' for the task.
## Setup/Requirements
* Python 3.x
* No external libraries required
## How to Run
1. Save the provided code in a file (e.g., `calculator.py`)
2. Open a terminal or command prompt and navigate to the directory where the file is saved
3. Run the calculator by typing `python calculator.py`
4. Follow the prompts to enter the first number, second number, and operation
5. The calculator will display the result and ask if you wish to perform another calculation
## Edge Cases
* Division by zero: The calculator will output a dramatic message expressing its 'horror' at being asked to perform such an 'atrocious' act, and then exit the program
* Non-numeric input: The calculator will output a dramatic message expressing its 'disgust' at the user's 'incompetence', and then exit the program
## Unit Tests
To run the unit tests, type `python calculator.py test` in the terminal or command prompt
