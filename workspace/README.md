# Reluctant Calculator
## Introduction
The Reluctant Calculator is a command-line calculator built in Python. It has a dramatic introduction and expresses reluctance to perform math operations. The calculator has four main functions: addition, subtraction, multiplication, and division.
## Setup/Requirements
- Python 3.x
## How to Run
1. Save the `app.py` file in your local workspace.
2. Open a terminal or command prompt and navigate to the directory where the file is saved.
3. Run the calculator by typing `python app.py` and follow the prompts.
## Features
- Dramatic introduction and farewell messages
- Four main functions: addition, subtraction, multiplication, and division
- Error handling for invalid user input and division by zero
## Usage
The calculator will ask for user input for the two numbers and the operation to be performed. It will then perform the operation and display the result, along with another dramatic message.
