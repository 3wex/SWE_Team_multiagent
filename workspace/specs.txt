The command-line calculator will be built in Python and will have the following features: 
1. The calculator will have a dramatic introduction when the program starts, expressing its reluctance to perform math operations. 
2. The calculator will have four main functions: addition, subtraction, multiplication, and division. 
3. Each function will be accompanied by a dramatic message expressing the calculator's disdain for the operation it is about to perform. 
4. The calculator will ask for user input for the two numbers and the operation to be performed. 
5. The calculator will then perform the operation and display the result, along with another dramatic message. 
6. The calculator will have error handling for invalid user input, such as non-numeric input or division by zero. 
7. The calculator will have a dramatic farewell message when the program ends.
